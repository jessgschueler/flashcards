JavaScript

JavaScript is a progrtamming language used to make web pages interactive.

Operators

Symbols used to perform operations on operands like values and variables.E.g, +,-,=, etc...

Variables

A container that stores information such as a number, object, or string.

Functions

A function is a bundle of code that performs a set of procedures.

Parameters

A placeholder variable that is included in a function definition. There is not a value for parameters, once the function is called an argument is passed into the parameter and gives it value.

Arguments

The value that is passed into a parameter, allows the function to perform its action.
